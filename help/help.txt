This is a guide to use the app

Requirement :

	> Internet is must and should for this app to work.

	> Proper atmosphere where there is less noise.

The program contains four display sections :

	a.This display shows whether or not the app has started (Shows if the app is dead or alive).

	b.This display shows app state whether app is listening to the user or recognizing the user's voice

	c.This display area shows the user command or input which was recognized by computer

        	d.This is the main display which shows output of the program

> Press 'Start' button on the bottom of the screen to start the application.Now display gets updated and shows
app has started on top of the screen.

> Press 'Stop' to stop the app anywhere in the middle.
(Note : It does not quits immediately because the thread is holding a critical resource that must be closed
 properly) 

> Press 'Restart' to Restart the app when app is running.
(Note : If app is stopped already then there is no meaning in pressing restart instead press start button.Hence
when app is already in stop state and you press 'Restart' button a warning is showed)

> Please speak only when the app shows 'Listening.....' otherwise if the app is in 'recognizing' state then it is
not going to take your command

> When app is in listening state speak the appropriate command only commands mentioned here will work.And 
questions must also be accurate.
